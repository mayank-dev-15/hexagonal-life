## About

**hexagonal-life** is a conway's game of life on hexagonal grid.

### Key Features

- Hexagonal grid rendering on canvas
- Click to toggle cells, drag to paint
- Play/pause, step, clear, randomize controls
- Multiple rule sets (Standard, HighLife, Day & Night)
- Hex-adapted preset patterns
- Age-based cell coloring
- Population and generation counters
- Zoom and pan navigation
- Export current state as image

---

## Tech Stack

- HTML5 Canvas 2D
- Vanilla JavaScript
- CSS3
- Cellular automata

---

## Controls & Usage

| Control | Action |
|---------|--------|
| `Click` | Toggle cell |
| `Click+drag` | Paint cells |
| `Space` | Play/pause |
| `Step` | Advance one generation |
| `R` | Randomize |
| `Scroll` | Zoom |
| `Arrow keys` | Pan |
